100% offline · open source · macOS

Talk. And it types.

Freest is a free, fully offline voice-dictation app for the macOS menu bar. Hold one hotkey, speak, and your words are typed right at the cursor — in any app. No accounts, no cloud, no API keys.

Get started → Set it up with your AI agent

One key to dictate

⌃+ ⌥+ D

Hold ⌃⌥D (Control-Option-D), speak, then release. Freest transcribes on-device and pastes the text at your cursor. That’s the whole thing.

Get started

Up and running in four steps

Freest is a small Swift app you build once. After that it lives in your menu bar.

1

Install

Clone the repo and run scripts/build-app.sh to produce Freest.app. Requires macOS 14+.

2

Grant access

On first launch, allow Microphone (to hear you) and Accessibility (to type at the cursor).

3

Get the model

Download the default base speech model once. After that, everything runs offline.

4

Press ⌃⌥D

Hold the hotkey, speak, release. Your words appear wherever your cursor is.

Why Freest

Private by design

Nothing you say leaves your Mac. The only time Freest touches the network is the one-time model download.

🔒

Fully offline

Recording, transcription, and cleanup all happen on-device. No accounts, no API keys, no telemetry.

⌨️

One-key dictation

Hold ⌃⌥D and talk. It pastes at the cursor in any app. Change the hotkey anytime in Settings.

✨

Smart cleanup

Optional local tidy-up fixes casing, punctuation, and filler words — or use on-device Apple Intelligence.

🎯

Menu-bar native

Lives quietly in the menu bar. No Dock icon, no windows in your way — just there when you need it.
